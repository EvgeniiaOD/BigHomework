## Описание блок-схемы ##
1. Есть исходный массив строк с элементами разной длины. Вводим переменную **count**.  Изначально присваиваем ей значение 0. Это будет счетчик элементов исходного массива, удовлетворяющих условию
 >длина которых меньше, либо равна 3 символам
 >
2. Запускаем цикл подсчета: проходим последовательно по всем элементам исходного массива и проверяем их на соответствие поставленному условию. Если условие выполнено, то  значение счетчика увеличиваеется на 1. Так мы определяем *длину нового массива*.
3. создаем новый массив строк, длина которого равна значению счетчика **count**. Вводим переменную **j** и присваиваем ей значение 0. Принимаем данную переменную за индексы нового массива.
4. Запускаем цикл заполнения нового массива: проходим последовательно по всем элементам исходного массива и проверяем их на соответствие условию
>длина которых меньше, либо равна 3 символам
>
Если условие выполнено, то записываем это значение в новый массив, увеличиваем индекс **j** на 1. Если условие не выполняется, то элемент в новый массив не записывается. 

5. Выходим из цикла либо, когда пройдем по всем элементам исходного массива, либо, когда новый массив будет полностью заполнен (количество подходящих элементов уже известно)

6. Выводим новый массив на печать.
